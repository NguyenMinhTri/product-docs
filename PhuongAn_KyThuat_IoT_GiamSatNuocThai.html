FTP Cloud Server + Phần Mềm Giám Sát Nước Thải Online

Hạng mục 1 Dành cho tủ quan trắc có Data Logger bên trong

Tủ quan trắc nước thải có tích hợp sẵn Data Logger bên trong. O&M - Navis sẽ cung cấp máy chủ FTP trên nền tảng đám mây để Data Logger đẩy dữ liệu lên, kèm theo phần mềm Web App giám sát trực tuyến cho phép xem dữ liệu và xuất báo cáo.

  • Cung cấp Server FTP Cloud — Máy chủ FTP trên nền tảng đám mây, cấu hình tài khoản FTP để Data Logger trong tủ quan trắc tự động đẩy dữ liệu lên theo chu kỳ
  • Cài đặt & cấu hình Data Logger — Thiết lập thông số kết nối FTP (IP, port, user/pass) cho Data Logger tại hiện trường, đảm bảo dữ liệu được truyền liên tục và ổn định
  • Phần mềm giám sát nước thải — Web App — Hiển thị dữ liệu trực tuyến qua trình duyệt web (PC, tablet, điện thoại):
    • Thông số nước thải: pH, COD, BOD, TSS
    • Biểu đồ theo thời gian thực
    • Lưu trữ dữ liệu lịch sử
  • Xuất báo cáo — Báo cáo theo ngày/tuần/tháng, xuất file Excel/PDF để lưu trữ và nộp cho cơ quan quản lý môi trường
  • Cảnh báo vượt ngưỡng — Thông báo tự động khi các chỉ số nước thải vượt ngưỡng cho phép
📶

Gateway Navis MOD + Phần Mềm Giám Sát Nguồn Điện

Hạng mục 2 Kết nối đồng hồ đa năng MFM383A qua Modbus

Cung cấp Gateway Navis MOD kết nối với đồng hồ đa năng MFM383A qua giao thức Modbus RTU/TCP. Dữ liệu điện năng được đẩy lên Cloud và hiển thị trên phần mềm Web App giám sát.

  • Cung cấp Gateway Navis MOD — Thiết bị chuyển đổi giao thức Modbus RTU/TCP sang Cloud, hỗ trợ kết nối WiFi / Ethernet / 4G
  • Kết nối đồng hồ đa năng MFM383A — Đọc dữ liệu qua Modbus:
    • Tổng công suất tiêu thụ (kW)
    • Điện áp 3 pha (V)
    • Dòng điện 3 pha (A)
  • Cấu hình & lập trình Gateway — Cấu hình địa chỉ Modbus, bảng thanh ghi (register map), tốc độ truyền thông (baud rate) cho đồng hồ MFM383A
  • Phần mềm giám sát nguồn điện — Web App — Hiển thị real-time các thông số điện năng, biểu đồ công suất tiêu thụ theo thời gian
🔧

Gateway Navis MOD + Phần Mềm Giám Sát Trạng Thái Thiết Bị (PLC)

Hạng mục 3 Kết nối PLC tủ điện chính qua Modbus

Cung cấp Gateway Navis MOD kết nối với PLC trong tủ điện chính qua giao thức Modbus RTU/TCP. Đọc trạng thái vận hành của hệ thống và hiển thị trên phần mềm Web App.

  • Cung cấp Gateway Navis MOD — Kết nối PLC trong tủ điện chính, đọc dữ liệu trạng thái vận hành qua Modbus RTU/TCP
  • Giám sát 3 trạng thái thiết bị —
    • 🟢 Đang chạy — Hệ thống hoạt động bình thường
    • 🟡 Dừng — Hệ thống đã dừng hoạt động
    • 🔴 Lỗi — Có sự cố, cần kiểm tra xử lý
  • Cấu hình & lập trình Gateway — Cấu hình địa chỉ Modbus, bảng thanh ghi trạng thái, mapping tín hiệu từ PLC
  • Phần mềm giám sát trạng thái — Web App — Hiển thị trạng thái real-time, mô hình hoạt động trực quan mô phỏng nguyên lý vận hành hệ thống
  • Cảnh báo thông minh — Thông báo ngay khi hệ thống gặp lỗi, thiết bị dừng bất thường, hoặc cần bổ sung hóa chất / dinh dưỡng

System Architecture Overview

✅ Tóm Tắt — O&M Cung Cấp

🔧 Phần Cứng

  • • Gateway Navis MOD (cho đồng hồ MFM383A)
  • • Gateway Navis MOD (cho PLC tủ điện)
  • • Server FTP Cloud (cho Data Logger tủ quan trắc)

💻 Phần Mềm

  • • Web App giám sát nước thải (pH, COD, BOD, TSS)
  • • Web App giám sát nguồn điện (V, A, kW 3 pha)
  • • Web App giám sát trạng thái thiết bị (PLC)
  • • Xuất báo cáo Excel/PDF

⚙️ Dịch Vụ Cài Đặt & Cấu Hình

  • • Cấu hình FTP cho Data Logger tủ quan trắc
  • • Cấu hình Modbus Gateway ↔ Đồng hồ MFM383A
  • • Cấu hình Modbus Gateway ↔ PLC tủ điện chính
  • • Hướng dẫn vận hành & bàn giao hệ thống

🛡️ Hỗ Trợ Kỹ Thuật
